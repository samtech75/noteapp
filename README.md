# Note App (MERN Stack)

A full-stack Note Application built using MERN Stack.

## Features
- Create, edit, delete notes
- JWT-based authentication
- Secure REST APIs
- Responsive React UI

## Tech Stack

### Frontend
- React
- Bootstrap
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Structure
- frontend/ → React application
- backend/ → Node + Express API

## How to Run
1. Clone the repository
2. Install dependencies in frontend & backend
3. Setup .env in backend
4. Run backend & frontend servers
